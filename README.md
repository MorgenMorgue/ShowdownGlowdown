ShowdownGlowdown is a customisable CSS plugin script for PokemonShowdown.com, please find the guide at https://www.youtube.com/@MorgenMorgue/videos

You will find all files by clickig the "main" tab and navigating the branches.

Alternatively, for people looking for the video-editing version, you can download the full mod onto your computer manually by downloading the .zip files.

If you have made your own Weather/Field effect pack and would like to share, please tweet them at @_MorgenMorgue
